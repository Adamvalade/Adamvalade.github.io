Projects

  • The Legend of Zelda (Unity Remaster)

This project was implemented in C# using the Unity game engine. My primary contributions included developing the health system (for both players and enemies), implementing the Goriya enemy, the boomerang weapom, building the UI system, and more. I also kept the user experience in mind throughout development, ensuring smooth gameplay and interface design, while contributing to general debugging and polish. The team followed an agile development methodology, using Jira to track tasks and progress.
